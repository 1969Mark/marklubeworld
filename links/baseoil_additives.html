API 基礎油分類

潤滑油的基礎油根據 API 分類可分為五大類。這些分類基於飽和烴、黏度指數和硫含量，主要反映了基礎油的氧化性能。

礦物油 (Group I, II, III)
  • 礦物油是最早被使用的基礎油，通常由原油經蒸餾和精煉過程製成
  • Group I：傳統溶劑精煉出來的基礎油，含硫量較高，氧化穩定性較差
  • Group II：氫裂解製程所煉出來的基礎油，含硫量較低，氧化穩定性較好
  • Group III：氫裂解過程與蠟油異構化製程所煉出來的類合成基礎油，具有高度的氧化穩定性
合成油 (Group IV)
  • PAO (聚α-烯烴) 合成油是100%合成的基礎油，具有卓越的性能
  • 在高溫或低溫下都能提供良好的潤滑效果，且具有高度的氧化穩定性
其他合成油 (Group V)
  • 包括酯類、矽氧樹脂、磷酸酯、PAG (聚亞烷基二醇) 等
  • 酯類油可改善潤滑添加劑的溶解性，提高潤滑油的使用壽命
  • 符合EAL規範的環保潤滑油(脂)

基礎油特性分析

基礎油與清淨性的關係

  1. 清淨性是測定發動機在活塞上沉積物的多少與狀態
  2. 沉積物是基礎油在活塞上因高溫氧化而形成，因此基礎油本身的氧化穩定性高就不易生成沉積物
  3. 飽和烴含量高沉積物就少，清淨性就好；但含量太高時清淨性反而差
  4. 飽和烴含量在80-90%之間最佳，而II、III類油飽和烴含量都在90以上，因而清淨性不如I類油（二衝程汽缸油來源為I類油）

基礎油與蒸發度的關係

  1. 對成品油的揮發度要求也就是對基礎油揮發度的要求，任何添加劑均無能為力
  2. 同一黏度而不同API分類的基礎油揮發度不同，當基礎油的黏度指數越高（從I類到IV類），則其揮發度越來越低
  3. IV類的PAO分子量分布比I、II、III類油都窄，因而揮發度低（narrow cut）

基礎油與低溫黏度的關係

  1. 任何黏度指數增進劑都會改善油品低溫黏度特性，因此只有基礎油的低溫黏度達到一定標準才能配製出合乎需要的油品
  2. 基礎油黏度指數越高，其低溫性能就越好（低溫時黏度不會過高，方便設備的啟動）
  3. 採用加氫處理製程可以生產黏度指數很高的II類、III類油，可配製所需的低溫低黏度油品

非引擎類潤滑油

添加劑所扮演的角色是補足基礎油本身不足的特性。非引擎類油以ISO在40°C為黏度規範：

齒輪油 (Gear Oil)
齒輪運轉時伴隨著撞擊與摩擦力道，故對添加劑的要求著重於抗摩擦與極壓特性。常用黏度等級：ISO VG 150、220、320、460、680。
液壓油 (Hydraulic Oil)
利用潤滑油的不可壓縮性傳遞作用力。添加劑著重於抗氧化特性。常用黏度：ISO VG 32、46、68。船舶用液壓油因低溫環境考量，一般會選用高黏度指數的液壓油。
空氣壓縮機油 (Air Compressor Oil)
有螺旋式(Screw type)與往復式(Reciprocating type)等種類。添加劑著重於抗氧化性。一般選用ISO VG 46、68；船舶往復式多段壓縮機選用ISO VG 100。新機建議用礦物油磨合期約300小時後再改用合成油。
冷凍機油 (Refrigeration Compressor Oil)
潤滑油須依照冷媒的種類而定，否則會有不相容的狀況。HFC環保冷媒（如R134a、R404A、R507）是現在與未來的使用趨勢，對應使用Polyolester類的合成油。黏度一般為ISO VG 46、68、100。絮點(Flock Point)必須低於蒸發器的溫度。

引擎類潤滑油

引擎類油以SAE在100°C為黏度規範：

低、中、高速引擎分類
  • 高速引擎（>1,200rpm）：汽車、救生艇等，規範API CG-4、CI-4、TC-W3
  • 中速引擎（300~1,200rpm）：發電廠、船上發電機，廠家：Everllence、Wartsila、Yanmar
  • 低速引擎（<300rpm）：船舶主機，廠家：Everllence、WinGD、MHI
二衝程與四衝程引擎潤滑設計

四衝程引擎：單一油品循環，黏度SAE 30或40，鹼值(BN)與燃料油含硫量相關。2020年後使用低於0.5%硫份燃油，建議BN 20。

二衝程引擎：兩套潤滑系統（汽缸油+系統油），以填料函(Stuffing Box)分隔：

  • 汽缸油：一次性潤滑，黏度SAE 50，BN等級有40、60、70、100、140
  • 系統油：黏度SAE 30，鹼值BN 5。可能因填料函磨損而被汽缸油污染
