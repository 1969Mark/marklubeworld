執行摘要

本簡報綜合分析了齒輪箱潤滑油選用、設備維護策略及失效診斷的關鍵要素。核心結論指出，現代齒輪設備管理正從傳統的定期維護轉向基於狀態的維護（CBM），此策略不僅能顯著延長設備壽命，更能滿足海事船級社對於延長檢驗週期的嚴格要求。

潤滑油的正確選用是設備可靠性的基石，其三大核心要素——基礎油、黏度與添加劑——共同決定了設備的運行效率與壽命。

一、潤滑油選用的核心三要素

潤滑油被視為齒輪箱的「血液」，其選擇與管理直接影響設備性能。摩擦學研究指出，大多數齒輪失效均可追溯至潤滑失效或選型不當。

1.1 基礎油類型：性能的基石

特性指標 礦物油 PAO PAG
API 分類 Group I / II Group IV Group V
黏度指數 (VI) 90 - 100 130 - 160 150 - 200+
氧化壽命 標準 長（2-3倍） 極長
抗微點蝕能力 一般 優良 卓越
與礦物油相容性 N/A 可混溶 完全不可混溶
典型應用 一般工廠 港口起重機、長換油週期 蝸輪蝸桿、風力發電

🏔️ 概念類比：登山運動員

  • 基礎油是「體質」：礦物油為普通體質，易疲勞；PAO/PAG為專業運動員，耐力強
  • 黏度是「登山鞋厚度」：慢走重載需厚底鞋；快速輕載需薄底鞋
  • 添加劑是「護具與營養品」：極壓添加劑如同護膝，防止極限壓力下受傷

1.2 黏度選擇：油膜厚度的關鍵

黏度是選油的首要參數，直接決定金屬接觸面間的油膜厚度。

高速輕載
應選用較低黏度（如 ISO VG 68, 100），以減少攪拌阻力與發熱
低速重載
必須依賴高黏度（如 ISO VG 320, 460, 680）來維持油膜厚度

Lambda 比 (λ) 與壽命

Lambda 比 潤滑狀態 磨損情況
λ > 3 全油膜潤滑 幾乎無磨損，設備壽命最長
1 < λ < 3 混合潤滑 部分金屬接觸，需依賴抗磨添加劑
λ < 1 邊界潤滑 極易發生磨損與膠合，完全依賴極壓添加劑

1.3 添加劑技術：極端工況的化學防護

  • 極壓 (EP) 添加劑：含有活性硫、磷化合物，在高壓高溫下生成保護膜。注意：活性硫對黃色金屬（青銅）具腐蝕性
  • 抗磨 (AW) 添加劑：如 ZDDP，主要在混合潤滑區提供保護
  • 防銹抗氧化 (R&O) 添加劑：適用於輕載且濕度高的場合

二、海事應用的特殊潤滑挑戰

在船舶應用中，水分是潤滑油的主要污染物，帶來多重威脅：

  • 降低油膜強度：水在高壓區會瞬間氣化，破壞油膜承載力
  • 引發腐蝕：特別是海水會導致嚴重的電化學腐蝕
  • 導致添加劑失效：部分添加劑遇水會發生水解

⚠️ 抗乳化性至關重要

海事齒輪油必須具備卓越的抗乳化性（Demulsibility）。DNV 等船級社明確要求推進系統齒輪油需通過嚴格的抗乳化測試（如 ASTM D1401，30分鐘內乳化層體積小於3ml）。

三、齒輪失效模式與標準化診斷

應採用 ANSI/AGMA 1010-F14《齒輪齒面磨損與失效術語》 規範進行標準化評估。

3.1 主要失效類型

失效類型 機制 特徵 對策
磨粒磨損 硬質顆粒刮削齒面 沿滑動方向的刮痕 改善密封與過濾
微點蝕 油膜厚度不足（λ < 1） 灰暗磨砂狀斑塊 使用 PAG 基礎油
宏觀點蝕 反覆接觸應力 肉眼可見凹坑 提高黏度、降低負荷
膠合 瞬時接觸溫度超過油膜極限 粗糙撕裂狀條紋 使用 EP 添加劑
彎曲疲勞 齒根應力集中 「海灘紋」斷口 設計優化、減少過載

四、維護策略與先進診斷技術

現代維護策略強調從被動修復轉向主動預測，核心是從「預防性維護」過渡到「狀態基礎維護 (CBM)」。

4.1 工業與海事實務維護

日常巡檢
檢查油位、油色、溫度、異音與呼吸器狀態
月度/季度任務
檢查過濾器壓差、執行振動分析、定期取油樣送檢
年度大修
雷射對中校正、內視鏡檢查與地腳螺栓緊固

4.2 先進診斷工具

診斷技術 原理 關鍵指標
振動頻譜分析 監測齒輪嚙合頻率及諧波 邊頻（Sidebands）增長是局部故障信號
磨粒分析 顯微鏡觀察油樣中磨損顆粒 球狀顆粒=疲勞；捲曲切屑=磨粒磨損
紅外線熱影像 掃描設備溫度分布 同類軸承溫差 > 5-10°C 需關注

五、核心結論與戰略建議

  1. 維護策略轉型：全面推行狀態基礎維護（CBM），以油樣分析和振動監測為核心
  2. 潤滑管理精細化：根據工況和環境精準選擇潤滑油，海事應用中抗乳化性與極壓保護性同等重要
  3. 檢查與診斷標準化：採用 AGMA 1010-F14 術語規範目視檢查，依據 ISO 20816 評估振動數據
  4. 確保合規性：嚴格遵循船級社檢驗規則，保存完整的狀態監控記錄
